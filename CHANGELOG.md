# 更新日志

## 1.0.8

- 发送视频前先下载到本地，兼容 Kook 无法直接读取 Seedance CDN 视频地址的问题。
- 增加下载失败和本地上传日志。

## 1.0.7

- 增加任务提交、轮询、终态和视频回传的详细日志。

## 1.0.6

- LLM Tool 改为提交任务后立即返回，避免超过 AstrBot 240 秒工具超时。
- 视频完成后由插件后台主动发送结果。

## 1.0.5

- 增加当前激活人设和人设列表配置。
- 没有传图片时自动使用当前人设参考图。
- 自动把人设基础描述加入视频提示词。

## 1.0.4

- LLM Tool 参数为空时自动从当前消息提取视频提示词。
- 自动读取当前消息中的图片作为视频首帧。
- 增加更新日志文件。

## 1.0.3

- 修复 AstrBot 事件参数注入导致的序列化错误。
